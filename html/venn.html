Venn diagram¶

How to make easy venn (or euler) diagrams of taxa shared or unique to a group of samples

In [2]:
# Package with venn functions
library(MicEco)
In [4]:
# Data
library(phyloseq)
load("../data/physeq.RData")

With counts of shared and unique taxa (ASVs):¶

In [5]:
ps_venn(phy, group = "Time", type = "counts")

With percentage of shared and unique:¶

In [6]:
ps_venn(phy, group = "Time", type = "percent")

As an euler diagram¶

In [7]:
ps_euler(phy, group = "Time", type = "percent")

Only include ASVs present in at least 5% of the samples in each group:¶

In [9]:
ps_euler(phy, group = "Time", type = "counts", fraction = 0.05)

Percent is weighted by relative abundance¶

In [10]:
ps_venn(phy, group = "Time", type = "percent", weight = TRUE)

Changing colors etc.¶

In [11]:
ps_venn(phy, group = "Time", fill = c("red", "green", "blue"), alpha = 0.5)
